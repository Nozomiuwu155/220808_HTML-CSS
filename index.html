CSS套用




指令 功能
font-size : 50px 調整文字大小（px, rem）
font-style : italic 調整文字斜體（italic, oblique）
letter-spacing : 25px 調整文字距離（px, rem）
line-height : 150px 行高度（px, rem）
text-shadow : 5px 5px 0px #ffffff 文字陰影（px, rem）
text-align : center 設定文字位置（center, left, right）
color : #red 調整元素的顏色
background-color : #ffffff 調整元素的背景顏色
border : 3px solid #black 設定元素邊框（px, rem / solid, dotted, dashed）
background-image : url("圖片網址") 設定背景圖片
height : 70px 調整元素的高（px, rem）
width : 300px 調整元素的寬（px, rem）
list-style-type : circle 清單造型（disc, circle, square...）
margin : 50px auto 調整元素外部的邊距（外部邊界）（px, rem）
（auto會置中）
padding : 15px 調整元素內部的邊距（內部邊界）（px, rem）
display : block 改變物件屬性
